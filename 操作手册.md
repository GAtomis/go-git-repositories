## 操作远程仓库
```shell
# clone 

git clone root@43.143.244.182:/root/git-test/hello.git
# 添加远程仓库
git remote add  origin root@43.143.244.182:/root/git-test/hello.git
# 推送本地代码到远程仓库
git push -u  origin master


```
